* when a mouse press is received on the tab label, or the body of an agent tab, the keyboard input focus should move to the command bar. this should happen even when the focus is being shifted from another application window in the operating system window manager to the janissary application window. For example, if a user is on the iterm application then clicks onto the janissary application over the tab label, the window focus will switch to the janissary application and then the mouse click will be processed by janissary, causing the hidden tab to be exposed and the keyboard input focus should move to the command bar.
