# Integrations and Soft Dependencies

Janissary works alongside optional tools that complement its functionality. These are called soft dependencies — tools that integrate well with Janissary but are not required to be installed or used.

## Supported Soft Dependencies

### Claude Code

An autonomous AI coding harness focused on single-task work. Users can spawn Claude Code tabs within Janissary or use it standalone for quick isolated tasks. Claude Code's strength is autonomous execution of one goal; Janissary's strength is coordinating multiple goals in parallel.

### Codex

An AI-assisted code editing experience for interactive pair programming. Codex helps users write code interactively in their editor, while Janissary runs agents autonomously in the background. The tools complement each other in different phases of the development workflow.

### OpenCode

An open-source framework for building AI-assisted coding tools. OpenCode-based tools can be integrated as agents within Janissary, or Janissary can coordinate with OpenCode-based custom tools via tabs and messaging.

## Documentation

Users can learn about soft dependencies and how to integrate them into their workflow in the user documentation under "Integrations."
