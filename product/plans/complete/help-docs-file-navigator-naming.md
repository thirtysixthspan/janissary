# Update help.md to say "file navigator" instead of "file tree"

**Complexity: 2/10** — two line edits in a single file; no code, spec, or asset changes.

## Goal

`help.md` still describes the `files` command and its tab controls using the old "file tree" name. The rest of the product (the docs site, recent plan filenames) has already moved to calling this feature the "file navigator." Bring `help.md` in line with that naming.

## Background

A prior technical-debt item asking for a full "file tree" → "file navigator" rename across backend code, wire-protocol types, ~20 web components, CSS, ~15 spec files, `help.md`, `CHANGELOG.md`, and public documentation was deferred as too complex (9/10). This item is a narrower, already-split-out piece of that work covering only "documentation."

Checking scope:
- `documentation/user-documentation/tab-types/file-navigator.md` already uses "file navigator" throughout its prose; the only "file tree" strings left there are the `/screenshots/file-tree.png` and `/screenshots/file-tree-sidebar.png` asset paths, which are generated by `scripts/docs-screenshots/manifest.mjs` — out of this task's allowed scope (`scripts/` isn't in the allowed file set), so those stay as-is.
- `README.md` and `documentation/developer-documentation/**` have no "file tree" mentions.
- `CHANGELOG.md` mentions are historical entries describing code identifiers as they were at the time (`FileTreeManager`, PR titles, etc.) — a historical record, not current-state documentation, so left unchanged.
- `product/specs/file-tree-tab.md` and its cross-references are functional specs, a separate category from documentation in this project's structure, and were called out separately in the original deferred item's complexity note — out of scope here.
- That leaves exactly two lines in `help.md`.

## Approach

Edit the two lines in `help.md` that name the feature "file tree" to say "file navigator" instead, matching the phrasing already used in `documentation/user-documentation/tab-types/file-navigator.md`.

## Implementation

1. **`help.md:31`** — `` `files` `` row: change `opens a file tree tab rooted at the issuing tab's cwd, or at `path`` to `opens a file navigator tab rooted at the issuing tab's cwd, or at `path``.
2. **`help.md:87`** — section heading: change `**File tree tab controls** (active only while a file tree tab is focused):` to `**File navigator controls** (active only while a file navigator is focused):`.

## Tests

None — `help.md` is prose documentation with no test coverage, and no behavior changes.

## Out of scope

- Renaming the `/screenshots/file-tree*.png` assets or `scripts/docs-screenshots/manifest.mjs`.
- Any change to `product/specs/file-tree-tab.md` or other spec files.
- `CHANGELOG.md` historical entries.
- The CSS backlog item, or the deferred backend/frontend code rename.
