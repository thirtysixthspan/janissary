Janissary design tenants

## Workflow support

Support your work through customized workflows, whether that is developing code, writing a book, or managing a business.

## Integrated agents 

Agents work along with you, to support your work.

## Agent orchestration

Agents work for you at your direction. Run multiple AI agents in parallel, interactively or automatically through scheduling.

## This is an application, not a platform

This tool is a user interface run on one system that is used to control work done on that system or others. It will not depend on any seperate service to run.

## Keyboard first interface design, mouse augmented

Core functionality is designed to work though keyboard interactions via text input or keybindings. Core functionality may be augmented by mouse input.  

## Identical control of local and remote resources

It should not matter where compute resources reside. All interactions should remain the same.

## Revision control of assets

Revision control of assets is achieved through local (git) and distributed (github) tooling.  

## Traceability of work

It's not magic, it's just complicated. Logs and reporting are the way to unravel that complexity when it all goes sideways on you.
